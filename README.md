# AI & Machine Learning Projects

A curated collection of end-to-end data science, machine learning, and artificial intelligence projects. This repository covers a wide spectrum of domains, from fundamental data analysis to advanced Generative AI and Computer Vision applications.

## 🚀 Projects at a Glance

### 1. [Python Foundations (FoodHub)](./Python%20Foundations/)
- **Objective**: Analyze order data for a NYC-based food delivery service to optimize operations.
- **Key Techniques**: Exploratory Data Analysis (EDA), Revenue Pattern Analysis, Customer Segmentation.
- **Outcome**: Identified top-performing cuisines and provided actionable insights to improve delivery efficiency.

### 2. [Machine Learning (Personal Loan Prediction)](./Machine%20Learning/)
- **Objective**: Predict which bank customers are likely to accept personal loan offers.
- **Key Techniques**: Classification, Decision Tree Classifiers, Hyperparameter Tuning (CCPs Alpha), Class Weighting.
- **Outcome**: Developed a high-recall model to assist marketing departments in target identification.

### 3. [Advanced ML (EasyVisa)](./Advanced%20ML/)
- **Objective**: Build a predictive model to help shortlist candidates with a high probability of visa approval.
- **Key Techniques**: Feature Engineering, Multi-model training (Random Forest, XGBoost), Recall Optimization.
- **Outcome**: Streamlined the visa portal process for better resource allocation.

### 4. [Intro to Neural Networks (ReneWind)](./Intro%20to%20Neural%20Networks/)
- **Objective**: Predictive maintenance for wind turbine generators based on sensor data.
- **Key Techniques**: Shallow Neural Networks, Adam Optimizer, Dropout Regularization, Batch Normalization.
- **Outcome**: Robust failure detection system focused on minimizing operational downtime.

### 5. [Intro to Computer Vision (HelmNet)](./Intro%20to%20Computer%20Vision/)
- **Objective**: Detect safety helmet compliance in real-time workspace environments.
- **Key Techniques**: Convolutional Neural Networks (CNN), Data Augmentation, Transfer Learning.
- **Outcome**: Automated PPE monitoring system to enhance industrial safety standards.

### 6. [Model Deployment (SuperKart)](./Model%20Deployment/)
- **Objective**: Forecast retail sales and deploy the model for business use.
- **Key Techniques**: Regression, XGBoost Regressor, Flask API Development, Model Serialization (Joblib).
- **Outcome**: Real-time forecasting tool accessible via a web endpoint.

### 7. [Generative AI & NLP (Healthcare RAG)](./Natural%20Language%20Processing%20with%20Generative%20AI/)
- **Objective**: Assist medical professionals with an evidence-based medical Q&A system.
- **Key Techniques**: RAG (Retrieval-Augmented Generation), LangChain, Vector Databases (ChromaDB), LLMs (Mistral-7B).
- **Outcome**: Accurate, citation-backed medical retrieval system leveraging the Merck Manuals.

## 🛠️ Technology Stack
- **Languages**: Python
- **ML/DL**: Scikit-Learn, TensorFlow, Keras, XGBoost
- **GenAI**: OpenAI GPT, LangChain, ChromaDB, HuggingFace
- **Deployment**: Flask, Joblib
- **Data & Viz**: Pandas, NumPy, Matplotlib, Seaborn
